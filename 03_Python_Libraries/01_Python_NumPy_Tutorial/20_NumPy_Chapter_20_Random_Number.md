
# NumPy Hands-On Tutorial

## Chapter 20: Random Number Generation (NumPy)

Random numbers are fundamental in data science, machine learning, simulations, statistical modeling, and scientific computing. Many algorithms rely on randomness for tasks such as:

- Sampling data  
- Initializing model parameters  
- Monte Carlo simulations  
- Shuffling datasets before training models  
- Generating synthetic datasets  

NumPy provides a powerful module for random number generation:

```

numpy.random

````

This module contains functions for generating random numbers, sampling data, and shuffling arrays.

Modern NumPy versions (1.17+) introduced a new random API using:

```python
np.random.default_rng()
````

This is faster and more reliable.

---

## Random Number Generation Basics

Random number generation means producing numbers that appear unpredictable and evenly distributed.

Computers actually generate **pseudo-random numbers**, meaning:

* Generated by deterministic algorithms
* Appear random but follow mathematical formulas
* Can be reproduced using a seed

---

## NumPy Random APIs

### Older API

```python
np.random.rand()
np.random.randint()
np.random.shuffle()
```

### Modern API (Recommended)

```python
rng = np.random.default_rng()
```

### Example

```python
import numpy as np

rng = np.random.default_rng()
print(rng.random())
```

**Output (example)**

```
0.73567281
```

---

## Generating Random Integers

Useful for selecting random indices, IDs, or categories.

### Function

```python
rng.integers(low, high, size)
```

### Parameters

| Parameter | Description               |
| --------- | ------------------------- |
| low       | minimum value             |
| high      | maximum value (exclusive) |
| size      | number of values          |

---

### Example: Single Random Integer

```python
import numpy as np

rng = np.random.default_rng()
num = rng.integers(1,10)

print(num)
```

**Output (example)**

```
7
```

---

### Example: Multiple Random Integers

```python
nums = rng.integers(1,100, size=5)
print(nums)
```

**Output (example)**

```
[23 89 45 10 67]
```

---

### Example: 2D Random Integer Array

```python
matrix = rng.integers(1,50, size=(3,4))
print(matrix)
```

---

## Generating Random Floats

Used in probability simulations and machine learning models.

### Function

```python
rng.random(size)
```

Generates values between:

```

```

---

### Example: Single Random Float

```python
rng = np.random.default_rng()
```

---

### Example: Multiple Random Floats

```python
nums = rng.random(5)

```

### Example: Random Float Matrix

```python
matrix = rng.random((3,3))
```

## Generating Random Arrays

Used to create synthetic datasets or initialize matrices.


## Random Sampling

Sampling means selecting random elements from an existing dataset.

### Function

```python
rng.choice()
```

---

### Example: Random Choice

```python
import numpy as np

rng = np.random.default_rng()
arr = np.array([10,20,30,40,50])

sample = rng.choice(arr)
print(sample)
```

---

### Multiple Samples

```python
samples = rng.choice(arr, size=3)
print(samples)
```

---

### Sampling Without Replacement

```python
samples = rng.choice(arr, size=3, replace=False)
print(samples)
```

Elements will not repeat.

---

### Sampling With Probability

```python
samples = rng.choice(arr, size=5, p=[0.1,0.1,0.2,0.3,0.3])
print(samples)
```

Some values become more likely to appear.

---

## Random Permutations

Permutation means rearranging elements randomly.

### Function

```python
rng.permutation()
```

### Example

```python
import numpy as np

rng = np.random.default_rng()
arr = np.array([1,2,3,4,5])

perm = rng.permutation(arr)
print(perm)
```

Original array remains unchanged.

---

### Permutation of Numbers

```python
print(rng.permutation(10))
```

Generates shuffled numbers from 0 to 9.

---

## Random Shuffling

Shuffling rearranges elements **in-place**.

### Function

```python
rng.shuffle(array)
```

### Example

```python
import numpy as np

rng = np.random.default_rng()
arr = np.array([10,20,30,40,50])

rng.shuffle(arr)
print(arr)
```

Original array is modified.

---

### Shuffling Rows of Matrix

```python
matrix = np.array([
    [1,2],
    [3,4],
    [5,6]
])

rng.shuffle(matrix)
print(matrix)
```

Rows are shuffled.

---

## Setting Random Seed

Random numbers normally change each time a program runs.

A seed allows reproducibility:

* Same seed → Same results

Important for:

* Machine learning experiments
* Scientific simulations
* Research reproducibility

---

### Setting Seed

```python
import numpy as np

rng = np.random.default_rng(seed=42)
print(rng.random())
```

---

### Example Demonstration

```python
rng1 = np.random.default_rng(10)
rng2 = np.random.default_rng(10)

print(rng1.random())
print(rng2.random())
```

**Output**

```
0.77132064
0.77132064
```

---

## Practical Example: Dataset Simulation

```python
import numpy as np

rng = np.random.default_rng(42)

# Generate synthetic dataset
ages = rng.integers(18,60, size=10)
heights = rng.random(10) * 50 + 150

print("Ages:", ages)
print("Heights:", heights)
```
